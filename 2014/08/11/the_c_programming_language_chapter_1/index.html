The C PROGRAMMING LANGUAGE第一章笔记

8月 11 2014

在看The C PROGRAMMING LANGUAGE第一章，一下记录一下学习心得


ANSI C(American National Standards Institute C)，大部分概念已经清晰，快速阅读，多多记录书中出现的计算机科学中经典的约定俗成的词汇

关键点：

  • ANSI C围绕6种数据类型：int, float, char, short, long, double，衍生出它们的arrays, structures, unions，以及它们的pointers，函数返回以上某些类型
  • The for is ususally appropriate for loops in which the initialization and increment are single statements and logically related, since it is more compact than while and it keeps loop control statements together in one place
  • magic numbers：记得在看linux文件系统时，看到过这个单词，当时想这个到底是什么number，本书第一章的symblic Constants这一节就介绍了magic number，从本书的天气度量转换的列子看，magic number就是一些让人不是那么容易知道意义的常量而已
  • text stream: a sequence of characters divided into lines; each line consists of zero or more characters followed by a newline character.
  • newline character: \n( backslash + n)
  • EOF 是在中定义的符号常量（symbolic constant），它的值跟任何一个char的值不同，所以当需要判断字符c是否和EOF相等时，c需要定义成int以便能够存放EOF值
  • 表达式（expression）是有值的，这样它又可以放入另外一个表达式中；statement就是表达式加分号
